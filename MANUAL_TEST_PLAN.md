## Подготовка окружения

- [ ] Деплой single-worker (`examples/single-worker`) на Cloudflare или локальный dev
- [ ] Открыть Playground: `http://localhost:8787/` или `https://<worker>.workers.dev/`
- [ ] Убедиться что health endpoint работает: `GET /health` → `200 OK`
- [ ] Telegram Bot Token настроен в env (если тестируем Telegram)
- [ ] Webhook зарегистрирован: `POST /telegram/webhook`
- [ ] KV namespace `SESSIONS` привязан

## 1. Playground: Simple Prompt Agent

> Агент без инструментов, только чат. Проверяем качество ответов, стриминг, историю.
