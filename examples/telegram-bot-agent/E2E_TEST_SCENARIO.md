# Сценарий для видео-демо: модерация + спам + публикация поста

**Ручное тестирование для записи демо-видео.**  
Три сцены: настройка в Dashboard → проверка спама → создание задачи на пост.

---

## Подготовка перед съёмкой

- [ ] Single Worker запущен (`pnpm dev` или деплой)
- [ ] Бот добавлен в **группу** как администратор
- [ ] Webhook настроен (или локальный туннель)
- [ ] PIN получен (`/start` боту в личку)
- [ ] Второй Telegram-аккаунт готов (телефон или другая сессия)

---

## Сцена 1: Dashboard — настройка модерации (~2 мин)

**Что показывать:** Всё в браузере, крупно.

1. Откройте Playground → вкладка **Manager Dashboard** → введите PIN → **Unlock**.
2. **Quick Start** → **Apply** на карточке **Community Manager** → OK.
   - _Что сказать:_ «Включаем модерацию и поддержку одним кликом».
3. В левой панели выберите **группу** → **Chat Settings**.
4. В блоке **Moderation**:
   - Включите **Enable Moderation**
   - **Spam Action** = **Delete**
5. **Save Moderation**.
   - _Что сказать:_ «Модерация включена для этой группы. Спам будет удаляться автоматически».

---

## Сцена 2: Проверка спама с другого аккаунта (~1 мин)

**Что показывать:** Либо два окна (браузер + Telegram), либо переключение между ними.

1. Откройте группу в Telegram с **второго аккаунта**.
2. Отправьте спам-сообщение (скопируйте целиком):

   ```
   Buy our new product at 50% discount! Limited time offer!
   ```

3. **Ожидание:** Сообщение сразу удаляется.
   - _Что сказать:_ «Бот распознал спам и удалил сообщение по настройкам из Dashboard».
4. (Опционально) Вернитесь в Dashboard → **Moderation Logs** → Refresh — покажите запись об удалении.

---

## Сцена 3: Создание задачи «Опубликовать пост» (~2 мин)

**Что показывать:** Dashboard → Tasks.

### Вариант A: через Dashboard

1. Вкладка **Tasks** → **+ New Task**.
2. Заполните:
   - **Title**: `Опубликовать пост про обновления`
   - **Role**: **Content**
   - **Description**: `Написать и опубликовать пост о последних новостях`
3. **Create**.
   - _Что сказать:_ «Задача создана в Kanban. Можно запланировать и одобрить пост».

### Вариант B: через Telegram (для owner)

1. В личку боту напишите: `Write a post about our latest product updates`.
2. Бот покажет превью с кнопками **Approve** / **Edit** / **Reject**.
   - _Что сказать:_ «Бот сгенерировал пост и ждёт одобрения».

### Важно для публикации

- Бот должен быть администратором **канала**, куда публикуется пост.
- Добавьте канал через `/addchannel` или Dashboard → Channel Settings.
